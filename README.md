### Requirements

`npm`

### Prepare

`npm install`

### Build

`npm run build`

### Run locally

`npm run dev`

### Run on production

Please do not use `npm` or `node` to deploy on prod. Expose the content of `dist` directory as static files directly under nginx or apache2/httpd.
